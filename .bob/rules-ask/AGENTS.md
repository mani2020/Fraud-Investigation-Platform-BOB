# Ask Mode Rules - Fraud Investigation Platform

## Project Context
Enterprise-grade real-time fraud investigation platform for banking payments.

## Technology Stack
- Java 17 with Spring Boot
- Kafka for event streaming
- PostgreSQL database
- React with IBM Carbon Design System

## Architecture Overview
- Event-driven microservices architecture
- Fraud agents as Java service classes (NOT ICA agents)
- Real-time payment processing via Kafka
- Explainable AI and fraud scoring algorithms
- React dashboard with IBM Carbon Design System
